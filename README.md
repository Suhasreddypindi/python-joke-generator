# Python Joke Generator

A simple Python application that fetches random jokes from a public REST API.

## Features

- Fetches random jokes from a live API
- Displays the setup first
- Waits for the user before revealing the punchline
- Lets the user request another joke
- Handles failed requests gracefully

## Technologies Used

- Python
- requests
- REST API
- JSON

## API Used

https://official-joke-api.appspot.com/random_joke

## How to Run

1. Install requests

```bash
pip install requests
```

2. Run

```bash
python joke_api.py
```
